INSTRUCTIONS FOR SETUP:
-----------------------

1. Install and activate Contact Form 7 plugin
2. Go to Contact > Contact Forms in WordPress admin
3. Create a new form or edit existing one
4. Copy the FORM CODE above into the Form tab
5. Configure the Mail settings using the MAIL SETTINGS above
6. Set up auto-reply using MAIL (2) settings
7. Save the form and note the shortcode (e.g., [contact-form-7 id="123" title="Contact form 1"])
8. Replace the shortcode in page-contact.php with your actual form shortcode
9. The CSS styling is already included in the template

SHORTCODE USAGE:
---------------

Replace this line in page-contact.php:
echo do_shortcode('[contact-form-7 id="contact-form" title="Contact Form"]');

With your actual shortcode:
echo do_shortcode('[contact-form-7 id="123" title="Contact form 1"]');
